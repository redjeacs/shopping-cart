# 🛒Mock Shopping Cart

A responsive, client-side e-commerce shopping cart web application built with **React** and bundled using **Vite** for instantaneous development speeds. 

This project explores frontend state management, dynamic item tracking, and interactive item configurations inside a simulated checkout ecosystem.

## 🚀 Features

- **Dynamic Product Grid**: Clean catalog displaying items with real-time detail rendering.
- **Interactive Cart Management**: Add items to the cart, dynamically scale item quantities, and safely remove items.
- **Live Price Calculations**: Automatic mathematical computation of item subtotals and the overall checkout total.
- **Lightning Fast Performance**: Structured on a streamlined React configuration with blazing-fast Vite Hot Module Replacement (HMR).
- **Exclusive Product Page**: Streamlined browser routing for individual shop categories.

## 🛠️ Built With

- **React** - Component-driven declarative UI architecture.
- - **React Router Dom** - Client-side declarative routing library.
- **JavaScript (ES6+)** - Dynamic cart state management and shopping logic.
- **CSS3** - Layout structuring, interactive effects, and storefront styling.
- **HTML5** - Application root mounting target.
- **Vite** - High-performance local development build tool.
- **ESLint** - Code quality consistency checker.

## 📦 Getting Started

Get a copy of this project running up and active on your local development server by executing these commands.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org) installed (v18.x or higher is highly recommended).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/redjeacs/shopping-cart
   cd shopping-cart
   ```

2. **Install all dependencies:**
   ```bash
   npm install
   ```

### Execution Scripts

Run the project locally using the configured automated scripts:

#### Local Development Server
Launches the ultra-fast Vite bundler locally:
```bash
npm run dev
```
Open your web browser and load `http://localhost:5173` to see the storefront.

#### Production Build Compilation
Compiles and completely minifies all assets into an optimized production-ready bundle inside the `dist/` directory:
```bash
npm run build
```

#### Test Suite Execution
Run tests using vitest on your code for component safety:
```bash
npm run test
```

#### Code Quality Check
Validates your JavaScript and component structures against formatting guidelines:
```bash
npm run lint
```
